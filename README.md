# Salesforce Cloud History Video

A local, file-driven motion design system for producing standalone technical explainer
clips. Each clip is a deterministic 1920×1080 HTML page designed for offline playback
and screen recording.

See [html_components/README.md](html_components/README.md) for architecture, authoring,
recording, and verification instructions.

## Quick Start

```bash
node html_components/verify.mjs
open html_components/01_concept_reveal/index.html
```
